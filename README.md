# Aether Deck

このリポジトリは、Aether Deck の配布ファイルを管理するためのリポジトリです。

## ダウンロード

最新のリリースは [Releases](https://github.com/k-aoshima/aether-deck-release/releases) ページからダウンロードできます。

## ファイル形式

- `.dmg` - macOS 用ディスクイメージ

## インストール方法

### macOS

1. [Releases](https://github.com/k-aoshima/aether-deck-release/releases) ページから最新のリリースを開く
2. `Aether Deck-0.1.0-arm64.dmg` をダウンロード
3. ダウンロードした `.dmg` ファイルを開く
4. アプリケーションを Applications フォルダにドラッグ&ドロップ
