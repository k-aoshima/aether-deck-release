# Aether Deck - リリース

このリポジトリは、Aether Deck の配布ファイルを管理するためのリポジトリです。

## ダウンロード

最新のリリースは [Releases](https://github.com/k-aoshima/AetherDeck/releases) ページからダウンロードできます。

GitHub Releasesから直接ダウンロードできるため、このリポジトリには配布ファイルは含まれていません。

## ファイル形式

- `.dmg.zip` - macOS 用ディスクイメージ（zip圧縮）

## インストール方法

### macOS

1. [Releases](https://github.com/k-aoshima/AetherDeck/releases) ページから最新のリリースを開く
2. `Aether Deck-0.1.0-arm64.dmg.zip` をダウンロード
3. zipファイルを解凍して `.dmg` ファイルを取得
4. ダウンロードした `.dmg` ファイルを開く
5. アプリケーションを Applications フォルダにドラッグ&ドロップ
